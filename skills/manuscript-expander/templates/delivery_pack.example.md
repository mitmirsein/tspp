# 전달 준비물 (Delivery Pack) — 믿음의 빈손

> 강단 직전 점검용. 시간·낭독·호흡. **모두 설교자가 자기 속도·발음으로 최종 보정한다.**

## 1. 시간

- 전체 추정 **≈ N분** (@320자/분 가늠 — 자기 녹음으로 보정).
- 목표 30분 대비: `under/on_target/over` 중 하나. *대략* 추정임을 잊지 않는다.

## 2. 섹션별 배분

| 섹션 | 추정 | 비중 | 메모 |
|---|---|---|---|
| 도입 | ≈0.7분 | 23% | |
| 논점 1 | ≈1.7분 | 62% | ⚠️ 쏠림 — 다른 논점과 균형 |
| … | | | |

- 한 섹션이 40%를 넘으면 배분을 점검한다(다른 논점이 눌리지 않게).

## 3. 낭독 보조 — 강단에서 *소리* 미리 정하기

**원어·외국어** (어떻게 발음/대체할지 미리 결정):
- `πτωχός` → 강단에선 음역("프토코스")보다 **풀어 말하기** 권장: "본문의 '가난'이라는 말은…". 원어를 굳이 소리 내면 회중이 멈칫한다.

**숫자·성구 읽기** (소리 내 읽을 형태로):
- `5:3` → "마태복음 **다섯 장 세 절**" · `1-12` → "1절부터 12절까지".

**호흡·분할** (읽다 숨찬 곳):
- 120자 넘는 긴 문장은 쉼표/마침에서 **끊어 읽거나 둘로 나눈다**. (해당 문장에 ‖ 호흡 표지를 단다.)

**영문 용어**: 회중에 맞춰 풀어 말할지 그대로 둘지 정한다.

## 4. 전달 메모 (설교자가 단다)

- 강조(↗)·쉼(‖)·느리게(~) 표지를 원고에 직접.
- 결단·기도 앞에서 호흡을 늦춘다.
